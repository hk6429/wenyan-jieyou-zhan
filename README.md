# 文言解憂站

國中、高中文言文閱讀理解練功站。字義／句義／段旨／篇章文意判讀四類題型，
搭配閃卡複習、PvE 對戰、「文豪錄」收藏養成，以及教師主持的全班即時文會。

## 核心功能

- 34 篇國、高中文言選文；實際篇目與分級以 `data/texts.json` 為準。
- 四類閱讀理解題型：字義、句義、段旨、篇章文意。
- 選文採「先讀後揭」：先找文本證據，需要時才開注釋，最後核對語譯。
- 首頁依錯題、到期複習與上次進度，只推薦一張可在五分鐘內完成的「下一步」。
- 每次作答保留信心、提示使用與錯因；錯題先反思，再揭露證據與近遷移任務。
- 「本次達標」與「穩固精通」分開，七日後通過平行題驗證才標記穩固精通。
- 求助錦囊會排除一個錯誤選項；混合練習平均涵蓋四類題型。
- 教師可從首頁直接開課，以班級碼及主持碼控制全班文會。
- 投影預設採安全模式，只顯示全班參與與整體答題分布，不公開末位或個別錯誤學生。
- 課後報表顯示各題答對率、未作答人數及待複習題，支援 CSV 與列印。
- 學生端不取得主持碼、答案表或全班名單；課後只顯示自己的成績。
- 合契達成學習證據後保證成功；即時排名對戰不套用裝備或文魄戰力加成。

## 本機預覽

```sh
npm run dev
```

瀏覽器開啟 `http://localhost:8080`。

## 驗證

```sh
npm install
npm run verify
```

`verify` 會依序執行資料品質檢查、全部單元／整合測試，以及瀏覽器煙霧測試。

## 資料結構

見 `data/texts.json`，每篇文本含 `passage`（原文全文）、`segments`（分段，供段旨題使用）、
`questions`（`type` 為 char/sentence/gist/theme 四種題型）。
